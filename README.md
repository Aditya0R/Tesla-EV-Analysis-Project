# Tesla EV Data Analysis

## Overview
This project analyzes Tesla EV deliveries and production data from 2015 to 2025.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Analysis Performed
- Data Cleaning
- Model-wise Analysis
- Region-wise Analysis
- Year-wise Trends
- Correlation Heatmap
- Machine Learning Prediction

## Dataset
Tesla EV Deliveries and Production Data (2015–2025)
